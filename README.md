# 🛡️ Pro-Level Discord Management Suite
**Optimized for Railway & GitHub | Slash Command Ready**

A high-end, button-integrated Discord bot designed for professional communities. Features a "Pro-UI" style with embeds and interactive elements.

### 🚀 Top Features
*   **🎫 Advanced Ticket System:** Button-based creation, staff-only private channels, and automatic category management.
*   **🔨 Core Moderation:** Full suite of slash commands (Kick, Ban, Clear) with built-in permission handling.
*   **👋 Welcome System:** Professional embed-style welcome messages for new members.
*   **💎 Pro UI/UX:** Clean embed designs, interaction buttons, and instant response systems.
*   **⚙️ Developer Ready:** Organized command handler, `.env` support, and 24/7 uptime stability.

### 📦 Setup
1. Upload to GitHub.
2. Connect to **Railway.app**.
3. Set your `TOKEN` in Variables.
4. Slash commands deploy automatically!
